# Find Lowest Common Ancestor in a Given Binary Tree
# Test cases :- 
    (i)
    n1 = 2 , n2 =  3
        1
    /  \
    2    3
    Output: 1
    Explanation:
    LCA of 2 and 3 is 1.

    (ii)
    n1 = 3 , n2 = 4
         5
        /
       2
     /  \
    3    4
    Output: 2
    Explanation: 
    LCA of 3 and 4 is 2.
